🛡️ หมวดหมู่: ความปลอดภัย (Security) - 8 Tests

TC-096: ทดสอบการหมดอายุของ Token
   • Token หมดอายุหลังไม่ใช้งาน

TC-097: ทดสอบการเข้าถึงหน้า Protected โดยไม่ Login
   • พยายามเข้า profile โดยไม่ login ควรถูก redirect

TC-098: ทดสอบการป้องกันหน้า Admin
   • user ทั่วไปไม่สามารถเข้าถึง /admin

TC-099: ทดสอบการจัดการ Session
   • Session ถูกสร้างและเก็บไว้ถูกต้อง

TC-100: ทดสอบการใช้หลาย Tab
   • เปิดหลาย tab แล้วทำงานได้ปกติ

TC-101: ทดสอบการลบ Session เมื่อ Logout
   • คลิก logout แล้ว session ถูกลบ

TC-102: ทดสอบการป้องกัน SQL Injection
   • กรอก SQL code ไม่สามารถ hack ได้

TC-103: ทดสอบการป้องกัน XSS Attack
   • กรอก script code ไม่สามารถ execute ได้
